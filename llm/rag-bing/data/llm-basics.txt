大模型基础概念

1. Transformer 架构
Transformer 是当前大语言模型的基础架构，核心组件包括多头注意力机制（Multi-Head Attention）、前馈神经网络（FFN）、层归一化（LayerNorm）和残差连接。2017年由 Google 提出，论文 "Attention Is All You Need"。

2. 大模型的训练三阶段
- 预训练（Pretrain）：在海量文本上学习语言规律，目标是预测下一个 Token
- 监督微调（SFT）：用人工标注的数据让模型学会对话
- 强化学习对齐（RLHF/DPO）：让模型输出符合人类偏好

3. 常用开源模型
- LLaMA 系列（Meta）：最广泛使用的开源基座模型
- Qwen 系列（阿里）：中文能力优秀，支持多模态
- DeepSeek 系列（深度求索）：MoE 架构，推理能力强
- Mistral 系列（法国）：小参数但性能出色

4. Token 与上下文窗口
Token 是大模型处理文本的最小单位。中文约 1.5 个字符 = 1 Token。上下文窗口指模型一次能处理的最大 Token 数，从早期的 4K 发展到了现在的 128K-1M。

5. 幻觉问题
大模型会生成看似合理但实际错误的内容，称为"幻觉"。缓解方法包括 RAG（检索增强生成）、Prompt 约束、微调对齐等。
